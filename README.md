# CUSTOMER SHOPPING BEHAVIOR ANALYSIS 

### SQL for Business Intelligence  

## Executive Summary

This project analyzes customer transaction data to identify revenue drivers, evaluate discount effectiveness, measure subscription impact, and segment customer loyalty.

Using structured SQL queries, raw transactional data was transformed into actionable business insights to support data-driven strategy.

- Discount profitability  
- Subscription contribution  
- Product satisfaction drivers  
- Loyalty revenue impact
